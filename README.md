# Social Network
  
## Project Description
This is a node nosql forum for keeping track of your friends and their thoughts and also reactions.  You can add, delete create and update your a database of friends and and keep track of all your thoughts about them as well as react to the thoughts and lists of your friends.
 
## Installation
This application requires Mongoose and Express.
